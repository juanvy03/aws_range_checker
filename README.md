# AWS Range filter

## Description
Simple web application to check AWS ranges for S3 and Route53 Healthchecks.


## Requirements

Install with:

    pip install -r requirements.txt


or manually install:


    Flask == 2.0.0  

    requests == 2.25.1

    ipaddress==1.0.23
    
    uwsgi==2.0.19.1
